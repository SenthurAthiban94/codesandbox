Display

Display is used to make an element inline or block or it can be used to arrange its contents in a layout (Flex/Grid/Flow)

Display can be grouped into the following types

  • Outside - Determines the elements outer display type
    • inline - Element will be inline occupying space based on its content. Arranged in a single line
    • block - Element will occupy entire line and will have a new line break before and after the current element.
  • Inside - Determines the arrangement of its contents within the element.
    • flow-root - block root element, content according to inline or block element
