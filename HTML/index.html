HTML Elements
A HTML Element consist of the following.
  • Opening tag
  • Closing tag
  • Content - Optional
Examples: (with content) < div > Content < /div >

Examples: (without content) < br />, < hr />

There are two types of HTML Elements
  • Block Elements
  • Inline Elements
Block Elements
